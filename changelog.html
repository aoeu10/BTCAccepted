May 18 , 2025
New Filtering Features
  • Added Bitcoin Only toggle for filtering businesses that exclusively accept Bitcoin
  • Added Nostr filter to quickly find businesses with Nostr presence
  • Improved filter status display to show Bitcoin Only and Nostr filter status
  • Enhanced filter layout and spacing for better usability
Version 0.2.1 - UI Enhancement Update
May 9, 2025
Improved User Interface
  • Enhanced typography with Roboto Condensed font for section headers
  • Improved visual hierarchy in business details view
  • Added Markdown support for business descriptions
  • Updated section labels styling for better readability
  • Improved content formatting with consistent font families
Version 0.2 - Map View Update
May 5, 2025
New Map Features
  • Added interactive Map View for geographic browsing of businesses
  • Implemented business clustering for locations with multiple businesses
  • Added detailed business information panel in Map View
  • Created visual indicators for business locations and clusters
  • Added ability to click markers to view business details
  • Implemented automatic map bounds adjustment based on filtered businesses

  • Created placeholder for upcoming BTCMap features
  • Added attribution and links to BTCMap project

Version 0.1 - Initial Release
March 19, 2025
New Core Features
  • Launched the Bitcoin-Accepting Businesses Directory with initial business listings
  • Implemented basic search and filtering functionality (by category, location, and text search)
  • Added support for multi-category businesses
  • Created card and list view options for browsing businesses
  • Built detailed business profile pages with shareable links
  • Added support for displaying Bitcoin payment methods (on-chain and Lightning Network)
  • Included social media links for businesses
New Location Features
  • Integrated location-based search functionality
  • Added support for sorting businesses by distance from selected location
  • Implemented "Use My Location" feature for geolocation-based searching
  • Created location dropdown with major cities worldwide
New User Interface
  • Designed responsive layout for all device sizes
  • Added category counts in filter dropdown menus
  • Implemented filter status display to show active filters
  • Created "About," "Contact Us," and "Links" information pages
  • Added "Clear Filters" option for easy reset
  • Improved list view with website and social media icons
  • Reorganized category structure for better organization
  • Added visual indicators for website and social media links
Improved Category Management
  • Moved "Bitcoin Hardware" to individual categories section
  • Added "Beauty/Cosmetic" and "Personal Care" as subcategories under "Health & Fitness"
  • Streamlined category selection interface
